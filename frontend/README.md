# MEW2 Frontend

## Authentication & Admin Setup

### First-Time Admin Signup
The application requires an initial administrator to be created.
1. Set the `VITE_ADMIN_SECRET` environment variable (e.g., in `.env` or Vercel/Netlify settings) to a secure string.
2. Sign up on the app using this secret as the "Invite Code".
3. The account will be created with `isAdmin: true`.

### Disabling Admin Secret
**Important:** After creating your first admin account, you should disable the secret creation path to prevent unauthorized admin access.
1. Remove `VITE_ADMIN_SECRET` from your environment variables.
2. Redeploy the application.
3. New users will now require an invite code generated by an existing admin.

## Development

Run `npm run dev` to start the development server.
